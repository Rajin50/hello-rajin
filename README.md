# Rajin Ibna Rajuanur Rahman - SQA Engineer Portfolio

A modern, responsive portfolio website for an aspiring Software Quality Assurance Engineer, built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- 🎨 **Purple-themed design** with deep purple primary and neon violet accents
- 🌙 **Dark mode by default** with smooth animations
- 📱 **Fully responsive** for all devices
- ♿ **Accessibility-focused** with ARIA labels and proper contrast
- 🚀 **SEO optimized** with Next.js
- 💫 **Smooth micro-interactions** using Framer Motion
- 📊 **QA-inspired aesthetic** emphasizing clarity and structure

## Sections

- Hero with animated background
- Technical Skills (categorized)
- Key Projects with GitHub links
- About Me (Education & Experience)
- Certifications
- Coding Profiles (LeetCode, HackerRank)
- Awards
- Contact Form

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons
- **SEO:** Next SEO
- **Language:** TypeScript
